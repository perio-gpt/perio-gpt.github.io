# periogpt.it

Benvenuto nel repository **perio-gpt.github.io**!  
Questo progetto ospita il sito web ufficiale di PerioGPT, dedicato all’intelligenza artificiale applicata all’odontoiatria parodontale.

## Contenuti del repository

- Sorgente del sito web, generato tramite tecnologie e linguaggi web (ad esempio HTML, CSS, JavaScript)
- Risorse statiche, immagini e file di configurazione
- Approfondimenti e documentazione tecnica correlata all'utilizzo e allo sviluppo del sito

## Come iniziare

1. **Clona il repository**
    ```bash
    git clone https://github.com/perio-gpt/perio-gpt.github.io.git
    ```
2. **Apri la directory**
    ```bash
    cd perio-gpt.github.io
    ```
3. **Visualizzazione locale**
    Apri il file `index.html` nel tuo browser per visualizzare il sito in locale.
